This tool visualizes the intersection between development feasibility, yard standards and parking requirements in El Paso, Texas.

To develop this, The Table of Density and Dimensional Standards (Appendix B) and Chapter 20.14 (Off-Street Parking and Loading) were ingested to create the app. Users can adjust the building and lot size to determine the feasibility of their proposal using the constraints that are specified for certain zoning districts and uses.

For this tool, the model focuses on multifamily structures.

Check out the appplication at: https://hoffmanap.github.io/testfit/
